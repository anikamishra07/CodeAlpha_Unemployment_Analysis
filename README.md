# Unemployment Analysis Using Python 📊

## Project Overview
This project analyzes unemployment trends in India using Python and data visualization techniques.

The objective is to understand unemployment patterns across different regions using Exploratory Data Analysis (EDA).

## Dataset
The dataset contains unemployment information with features like:

- Region
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Labour Participation Rate
- Area

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Project Workflow
1. Imported dataset
2. Cleaned the data
3. Checked missing values
4. Performed Exploratory Data Analysis
5. Created visualizations
6. Generated insights

## Conclusion
The analysis helped identify unemployment trends across different regions of India and compare unemployment patterns.
